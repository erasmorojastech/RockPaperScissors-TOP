## ✨ Overview
A simple implementation of the classic Rock Paper Scissors game, built as part of The Odin Project curriculum. The game is played entirely in the browser console, with the player competing against the computer over 5 rounds.

## 🎨 Design
No graphical user interface (GUI) was required for this project; the game runs in the console. The focus was on JavaScript fundamentals, including functions, conditionals, and basic game logic.

Instructions may be found in:
[Rock Paper Scissors - The Odin Project](https://www.theodinproject.com/lessons/foundations-rock-paper-scissors)

## ✨ Features
- Play 5 rounds of Rock Paper Scissors against the computer.
- Score tracking for both player and computer.
- Round-by-round results displayed in the console.
- Final winner announcement after all rounds.
- Case-insensitive input (e.g., "Rock", "ROCK", "rock" all accepted).

## 🚀 How to Use
1. Clone the repository
   ```bash
   git clone https://github.com/erasmorojastech/RockPaperScissors-TOP
   ```
   
2. Navigate to the project folder
3. Open `index.html` in your web browser
4. Open the browser console (F12 or right-click → Inspect → Console) to play the game. Follow the prompts.
